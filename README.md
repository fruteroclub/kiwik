# kiwik - Juega, Cultiva, Impacta

Bienvenido a kiwik, una plataforma que transforma ideas en impacto a través de juegos y comunidades. Este proyecto utiliza **Next.js**, **shadcn/ui**, y está diseñado para crear experiencias interactivas y colaborativas.

## Características Principales

- **Comunidades Vibrantes** - Únete o crea tu propia comunidad de innovación
- **Retos y Recompensas** - Gana $1UP y Regen Points participando en desafíos
- **Regenmons** - Colecciona y evoluciona tus criaturas digitales
- **Colaboración** - Trabaja con otros builders en proyectos de impacto
- **Microfinanciamiento** - Accede a $REGEN para impulsar tus iniciativas

## Tecnologías

- **Next.js** con App Router para estructura y enrutamiento óptimo
- **Shadcn** para componentes UI hermosos y reutilizables
- **Bun** como runtime para builds más rápidos
- **Dynamic Wallet** para manejo de carteras digitales
- Integración con blockchain para tokens y NFTs

## Inicio Rápido

### Prerequisitos

- [Node.js](https://nodejs.org/)
- [Bun](https://bun.sh/docs/installation)
- [Git](https://git-scm.com/)

### Instalación

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/kiwik/kiwik-app.git
   cd kiwik-app
   ```

2. **Instala las dependencias:**

   ```bash
   bun install
   ```

3. **Configura las variables de entorno:**

   Crea un archivo `.env` en la raíz del proyecto:

   ```plaintext
   NEXT_PUBLIC_DYNAMIC_API_KEY=tu_api_key
   NEXT_PUBLIC_RPC_URL=https://rpc-url
   ```
